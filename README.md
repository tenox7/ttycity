# ttycity — Micropolis (SimCity) for the terminal

A terminal / text-mode port of [Micropolis Activity](https://github.com/SimHacker/micropolis/tree/master/micropolis-activity) using [ncurses](https://en.wikipedia.org/wiki/Ncurses). Uses the original simulation engine as-is. Replaces just the X11/Tcl/Tk with ncurses UI.

## Running

All assets — the string/message tables, the scenarios and the bundled example
cities (originally from Micropolis Activity, so they are GPL) — are baked into
the binary (see `headers/res_data.h`, generated by `genres.c`).  `ttycity` is
therefore self-contained and needs no external files at runtime:

```sh
./ttycity                       # start a new random city
./ttycity ../cities/bruce.cty   # load a city file from disk
./ttycity -theme grass          # tan (default) | grass | dark land color
```

The built-in scenarios and example cities are on the in-game *New Game* and
*File → Load Built-in* menus; *File → Load from Disk* (or a `.cty` path
argument) loads a city from the filesystem, and any city can be saved back to
disk.  The `res/` and `cities/` directories in this repo are only the source
for the baked-in header (regenerate it with `make resdata`).

## Playing

The screen has a menu bar on top, a tool palette down the left (with the current
tool, cost, RCI demand and speed), the scrollable city in the middle, an
optional overview map on the right, and a status line at the bottom.

| keys | action |
|------|--------|
| arrows / `hjkl` | move the cursor |
| Shift + letter  | pick a tool (`R`es `C`om `I`nd, r`O`ad rai`L` `W`ire, `B`ulldoze `Q`uery, `F`ire police-`D`… see palette) |
| Enter           | build the current tool at the cursor |
| `m`             | cycle the overview map / data overlays |
| space           | pause · `0`–`3` set speed |
| Esc / F10       | open the menu bar |
| `g`             | new random city · `q` quit |

Menus cover New Game / scenarios, Load & Save, the Budget, City Evaluation,
History graphs, disasters, and options (auto-budget, sound, land color, …).
The interface is resizable and works down to 80×24.
